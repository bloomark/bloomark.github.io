7.a.y The value of a product being proportional to the number of users.

What are positive network effects?

A product has (positive) network effects if the value of a product to a user increases with the total number of users.
Example-Fax Machine – the value of a fax machine is proportional to the number of users.

7.b.z Yes, but only if the search engine learns from users and improves the product

Do search engines on the Web have network effects?
Yes search engines have network effects only if they are learning from customers. This is an indirect network effect. The more the search engines learn, the better would be the product.
